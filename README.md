# ZeroUpload

> Convert files without uploading a thing.

ZeroUpload converts your files **100% inside your browser**. No upload, no
server, no signup, no limits — and it even works offline. Your files never
leave your device.

## Why it's different

Other converters upload your private files to their servers (which is why they
have limits and paywalls). ZeroUpload does the entire conversion on **your**
device using the Canvas API and WebAssembly. Zero server cost = unlimited and
free, forever.

## Tech

Astro · React (islands) · Tailwind CSS v4 · Canvas API · Cloudflare Pages
